php-redis-install

2017-09-30
  • 需要phpize、php-config,安装方法：yum install -y php-devel
  • wget https://github.com/phpredis/phpredis/archive/master.zip
  • 或https://github.com/phpredis/phpredis/archive/develop.zip
  • unzip master.zip && cd **
  • phpize
  • ./configure -with-php-config=/usr/bin/php-config
  • make
  • sudo make install
  • cd /*/php.ini(查看php.ini位置用命令：php –ini)
  • 在Php.ini最后面添加extension=redis.so
  • 用Php -m 查看PHP安装的扩展模块
