centos7-nginx-环境准备

  • centos >= 7.0 64位

    uname;
    uname -r;
    uname -a;
    rpm -q centos-release
    cat /etc/redhat-release

  • cpus>=2；memory>=512m

  • ping www.baidu.com
  • yum list|grep gcc
    -
  • 关闭iptables规则

iptables -L;
iptables -F;
iptables -t nat -L
iptables -t nat -F

  • 停用selinux

    getenforce
    setenforce 0

  • yum -y install gcc gcc-c++ autoconf pcre pcre-devel make automake

  • yum -y install wget httpd-tools vim

  • sudo mkdir app backup download logs work

安装nginx

  • 添加nginx源–参看官网
  • yum -y install nginx
  • rpm -ql nginx —安装目录

配置

  • nginx -t；nginx -t -c
  • sudo service nginx start
  • sudo nginx -s reload
  • sudo nginx -s stop

[vagrant@localhost run]$ sudo nginx -s reload
nginx: [error] invalid PID number “” in “/var/run/nginx.pid”
nginx: [alert] could not open error log file: open() “/var/log/nginx/error.log” failed (13: Permission denied)
[vagrant@localhost run]$ sudo chmod -R 755 /var/log/nginx/
sudo nginx -c /etc/nginx/nginx.conf
sudo nginx -t
sudo nginx -s reload
