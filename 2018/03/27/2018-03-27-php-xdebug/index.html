xdebug的安装和配置

  1. 检查是否安装了xdebug

    通过phpinfo()和命令php -m查看

  2. 根据xdebug.org官网下载源码方式安装

    官方安装文档

    • 下载源码编译
      1. wget https://xdebug.org/files/xdebug-2.6.0.tgz
      2. tar -xzf xdebug-2.6.0.tgz
      3. cd xdebug-2.6.0
      4. phpize
      5. ./configure –enable-xdebug
      6. make
      7. sudo make install
